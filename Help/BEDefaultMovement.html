Default Movement Behavior

When in Base Edit Mode, having special behavior bound to your movement keys can be a hassle. Speed on Demand binds can get confused, for instance, leaving your toon flying into walls. Other movement overloads, like firing powers or combining inspirations, might be unwelcome while editing your base.

Checking this box will reset all movement keys to the game's simple default behavior while you're in Base Edit Mode. When you exit the mode, your Keybind Reset will be run, reconfiguring your movement keys to whatever you have set up for them via BindControl

This setting is recommended.
