Disable Chat Binds

The chat window is, by default, at the lower left of the UI, and overlaps the base editing UI. Hitting ENTER to start a chat, for instance, will pop up the chat window over the top of the base-edit UI and channel all keyboard events to the chat input box.

As this can be confusing and frustrating, this checkbox will tell BindControl to turn off all chat commands while you are in Base Edit Mode. The chat commands will be re-enabled when you leave the mode.

Some people enjoy chatting while building and might have their UI set up to facilitate that. Accordingly, this setting is completely a matter of preference.
