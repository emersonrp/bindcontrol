Disable Window Toggles

When the base editor starts up, the game toggles off all of the main windows, but it's possible to re-show them with their keybinds.

This can lead to cluttered and confusing behavior, especially if you've customized your UI to have some of those windows in nonstandard positions.

If this box is checked, BindControl will disable the binds that toggle the various game windows until you leave Base Edit Mode.

This setting is a matter of taste, but is usually safe to leave on unless you intentionally toggle various windows on and off while base editing.
