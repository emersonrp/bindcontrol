Base Edit Mode

This Bind Wizard allows you to create a keybind that will toggle you in and out of Base Editing Mode whenever you are in your supergroup base.

Toggling the keybind while in your supergroup base will activate the base editor, and optionally turn off various of your BindControl keybinds that you have set up that might be fiddly or in the way while in the base editor. For instance, Speed on Demand binds can get very confused while in the base editor, so the Bind Wizard has an option to disable these while you're in Base Edit Mode.

When you're done editing, you can toggle the keybind again, and you'll be dropped out of the base editor and your normal set of keybinds will be reloaded just as if you'd pressed the "Reset Binds" key.

Several base editing commands are available to be bound to keys for the duration of Base Edit Mode, including various toggles and item selection functions. These binds will be activated when you enter Base Edit Mode and deactivated when you exit.

Note that you have to have permissions to edit the base, and you have to be inside the base to edit it. Pressing the keybind outside of those conditions will probably put you in a weird state, but pressing it again to toggle back out should sort everything out.
