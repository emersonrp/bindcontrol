Binds Directory Location

  • Inside Game Directory - keybinds will be kept inside your game directory. This is the recommended setting, and is the default for new BindControl users.
  • Absolute Path - keybinds will be kept in a specified location in your filesystem. This is the traditional scheme for keybinds but can be harder to manage and maintain. Existing BindControl users will default to this setting to keep things as expected.

Changing this setting will require that you "Write Binds" on all Profiles to get the binds written to the new location. The old location will not be automatically cleared of keybinds.
