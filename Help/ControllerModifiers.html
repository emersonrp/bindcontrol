Extra controller modifiers were added in Homecoming Issue 27, Page 1. They can be used to make extra "chord"-style binds using a controller. The details of this scheme are too complicated to summarize here. Please check the Patch Notes for details.

Note that BindControl does NOT set up these modifiers for you. This configuration is intended to match whatever configuration you have manually added to the game with the /controller_modifiers and /extra_modifiers commands.
