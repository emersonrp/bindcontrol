Escape Configurator

By default, City of Heroes binds the Escape key to /unselect, which clears any target selection you might have. This is a reasonable thing for Escape to do, but over the years, we've come to expect more things from it.

With the Escape Configurator, you can assign the Escape key to do any number of interesting and useful things. This can help make City of Heroes feel more like modern games, though sometimes the effects can be a little fiddly and unexpected. Tinkering is encouraged.

The functions that this Wizard can bind to Escape are:

  • Unselect - clear any target selection you currently have, leaving you with nothing targeted. This is the default behavior.
  • Unqueue - unqueue / cancel any queued power. This can be useful if you find yourself changing your mind a lot about what to cast next, though having both this and unselect bound might be a bit frustrating.
  • Close All Windows - this will close all "non-essential" windows, leaving just the normal chat, nav, player, target, and power trays showing, depending on how you have your UI normally configured.
  • Exit Completed Mission - teleport out of a completed mission, equivalent to clicking the "Mission Complete" button.
  • Reset Camera - reset the camera to the default distance behind the player.
  • Choose No Reward - if a reward dialog is open, after a mission arc, for instance, this will close the dialog, choosing no reward.
  • Answer No to Dialog - equivalent to choosing "OK" or "No" or "Cancel" on any dialog being shown.
  • Show Main Menu - pop up the game's main menu. This could work well with Close All Windows or could be frustrating and annoying in that case. You do you.

Various combinations of these commands make better sense than others, but you can choose any or all of them. All that you choose will be bound to each press of Escape, so choose wisely, and experiment to discover what feels right.

This Wizard is a work in progress, and any and all feedback is welcome.
