Inspiration Combine Options

In this dialog, you can choose which types of inspirations the keybind will try to combine into the Target Type when the configured Key is pressed. The keybind will then try to use one of the Target Type inspirations on key release.

The binds will try to combine any available three inspirations of the Combine Types into one of the same level for the Target Type. All three levels of inspiration will be attempted.

Because of game limitations, not all Combine Types will fit into a single keybind. If there is too much to fit, BindControl will automatically create a set of rotating key binds that will make its way through the list on subsequent key presses, and finally wrap back around to the start. Key release will always attempt to use one of the Target Type inspirations.

If no Combine Types are chosen, the keybind will simply use one of the Target Type inspirations on keypress and do nothing special on key release.
