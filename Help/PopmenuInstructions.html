The Popmenu Editor helps you create, edit, import, and otherwise manage your installed popmenus.

You can use the buttons in the upper left to:

  • create a new popmenu from scratch
  • import a popmenu from a file
  • uninstall / delete a popmenu completely

The list on the lower left contains all popmenus installed to the menu directory. You can select any of the menus and test it using the "Test" button at the top.

Right-click on any entry in a popmenu to edit that entry or create a new one.

Menus shown in italics have not yet been loaded into BindControl. Selecting one will automatically load it. Menus shown in boldface contain unsaved changes.

Use the "Write Menu" button at the top to save your changes to a popmenu.

The "Generate Macro" button will create a simple macro command you can paste into the game to create a macro button that will pop up the popmenu.
