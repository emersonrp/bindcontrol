PowerBinder

PowerBinder is a powerful and flexible tool for creating keybinds with multiple commands or actions. It can, for instance, create a keybind that will change your costume, say something to your teammates, load a different BindControl profile, and change your window colors to match your new costume, all with a single press of a key.

It's also useful for simpler keybinds where the syntax of the slashcommand might be fiddly or complicated -- PowerBinder hides all of this complication behind simple dialog boxes and controls.

NOTE that City of Heroes only allows a single power activation per keybind. While PowerBinder will allow you to create keybinds that have multiple power activations in them, the game will only honor one of them, so the results might not be what you expect.

The Main Window

PowerBinder's main window features a list of "commands" that will be added to the keybind in order. You can add commands with the "Add Command" picker at the top, and can rearrange, edit, and delete existing commands using the buttons on the right. The read-only text box at the bottom shows the actual current contents of the keybind.

Commands

PowerBinder creates keybinds from a sequence of commands. A command typically consists of a single slashcommand plus the various options that the slashcommand might require. A given command might, for instance, specify a chat command, or toggle in and out of supergroup mode, or activate a given power. Each type of command has its own UI for configuring it, apart from a few very simple command types.

Some command types might add multiple slashcommands to the final keybind.

Current command types include:

Graphics / UI

Attribute Monitor
Activate the monitoring of one or more attributes in the game's Attribute Monitor Window.
Buff Display Settings
Change the configuration of the display of buff icons for self, group, and pets.
Graphics Settings
Change various settings related to graphics and rendering, including bloom, lod, scale, and others.
Screenshot
Take a screenshot in JPG or TGA format, optionally with or without the game's UI visible.
Performance Graphs
Show various in-game performance graphs, such as FPS or network performance.
UI Scale
Scale the entire game UI up or down.
Window Color
Set the game's global window color settings to a specified RGBA value.
Window Save / Load
Save or load window settings from a specified file.
Window Toggle
Toggle or set the visibility of a selected game window.

Inspirations

Use Insp By Name
Use an inspiration specified by name (Respite, Good Luck, etc).
Use Insp From Row/Column
Use an inspiration by activating a particular row and column in the inspiration tray, or the bottom-most inspiration in a given column.

Powers

Auto Power
Select the power that is being auto-fired on cooldown.
Power Unqueue / Abort
Cancel the currently queued power, or the currently queued power and the autopower.
Use Power
Use or cancel a power by name, optionally as a toggle. Available powers are selected based on the powersets defined in the General tab.
Use Power From Tray
Activate the power in a specific slot of a specific power tray.

Social

Away From Keyboard
Set the player AFK, with an optional message.
Chat Command
Send a message to a selected chat channel, optionally with custom chat colors.
Chat Command (Global)
Send a message to a specified chat channel (including custom ones created with /chat_create).
Costume Change
Change into a selected costume, optionally using a selected costume change emote.
Emote
Perform the selected emote.
Hide / Unhide
Choose various groups (friends, supergroup, tells, invites) to hide from or to cancel the hide condition; or open the hide dialog to manage this in-game.
LFG Settings
Manage your LFG settings, including queueing, your LFG status, and your search comment
SG Mode Toggle
Toggle "Supergroup Mode."

Targeting

Target Custom
Target something by name, specifying Near / Far / Next / Prev, as well as various categories (friend, alive, base, etc) to use for matching.
Target Enemy
Target an enemy by name, specifying Near / Far / Next / Prev.
Target Friend
Target a friendly by name, specifying Near / Far / Next / Prev.
Team/Pet Select
Select a specified team member or pet by number.
Unselect
Unselect any selected target.

Misc

Camera Commands
Various commands affecting camera distance, rotation, first vs third person, mouselook, etc.
Load Binds Directory
Load the reset.txt file from a selected binds directory. This can be used to change keybinds in-game to a different BindControl Profile, possibly in the case of changing specs. You can also create small "partial" profiles and load them without resetting all keybinds, allowing for complex and modular configurations of keybinds.
Movement Commands
Perform one of the standard movement commands. These are usually managed via the Movement Powers tab, but can be added to a PowerBinder bind if desired.

Custom

Custom Bind
Type your own slashcommand string as a command. This is useful for slashcommands not currently supported by PowerBinder, as well as, for instance, activating powers for unsupported archetypes on non-Homecoming servers.
