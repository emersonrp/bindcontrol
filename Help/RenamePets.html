Rename Pets Wizard

This wizard enables you to set up a set of unique names for your Mastermind pets, and assign them these names with a single keybind.

You'll probably only use this very very occasionally, but it's designed to simplify the process of getting unique names in-game to use with the Mastermind Page's "Select by Name" and "Bodyguard" binds.

The level slider is so that the binds will know which pets are available to be summoned and renamed. If you are above the max level of the slider, just keep it at the max.

Each pet name field will let you know, by turning red, if that pet name isn't unique enough for the "Select by Name" and "Bodyguard" binds to use correctly. Keep tinkering until you get a set of names that is adequately unique so that none of the text fields is red.

When you click "OK" on the dialog, the pet names on the Mastermind Page will update to match. If you edit the names there, the wizard here will update to match.

In-game: summon all of your pets before using the keybind! If you don't have all of your pets summoned, the results will be somewhat unpredictable.
