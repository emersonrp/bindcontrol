Custom Bind Wizard Binds

The Launch Custom Bind Wizard button will allow you to launch a Bind Wizard that will guide you through building a Custom Bind that performs some useful but complicated function.

Anything a Bind Wizard does could be handled manually using a Complex Bind, but the Bind Wizard makes it easier to create and maintain these more elaborate keybinds.

Available Wizards:
Escape Configurator
Configure how the Escape key behaves, choosing one or more commands to execute with each press.
Incarnate Set
Create a set of Incarnate Powers that will be slotted using multiple presses of the bound key.
Inspiration Combiner
Create a keybind that will consume certain inspirations on press, and combine other inspirations to make more on key release.
