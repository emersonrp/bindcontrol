qwy PetMouse Binds

The PetMouse Binds are a simple but powerful set of Mastermind keybinds that mostly use mod keys plus the mouse to select and give orders to pets individually and in groups.

The binds can install and use an optional, but highly recommended, popmenu, which offers quick access to pet functionality using keys easily accessible from the WASD "home" position. You will need to have set up your Game Directory in the Preferences dialog for this to be installed.

You can set these binds to any keys or mouse actions you like, but the defaults are highly recommended, as they allow easy control of pets without removing your hands from the mouse and the WASD keys.

These binds were posted to the Homecoming forums by user qwy in 2020. More information can be found on the original forums post.
